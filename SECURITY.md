# Security Policy

## Supported Versions

Security fixes are applied to the latest published version of `EdexUi-2026` and the current development branch.

## Reporting a Vulnerability

If you discover a security issue:

1. Do not publish full exploit details immediately.
2. Prefer a private GitHub security advisory if the repository enables advisories.
3. If private advisories are unavailable, open a GitHub issue labeled `security` with limited public detail and request a private contact path.

Please include:

- affected version or commit
- operating system and desktop session
- reproduction steps
- expected impact
- logs or screenshots if relevant

We will try to acknowledge reports quickly and coordinate a fix before wider disclosure whenever possible.
